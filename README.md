# view-svg

vscode view-svg extension

## Usage

To use this extension, create a TypeScript or JavaScript file with the following format:

```ts
export default {
  icon1: "xxxxx.svg",
  icon2: "xxxx.svg",
  icon3: "xxxx.svg",
};
```

You can preview the SVG files by pressing `command+shift+p` and selecting `preview svg`.
